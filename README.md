# go-mesh
service mesh base on golang

![](https://travis-ci.com/lixiangyun/go-mesh.svg?branch=master)

## step1 clone

```
git clone https://github.com/lixiangyun/go-mesh
cd go-mesh
```

## step2 build 

### linux
```
./build.sh
```

### windows
```
build.bat
```

## step3 run 

### linux
```
./run.sh
```

### windows
```
run.bat
```
